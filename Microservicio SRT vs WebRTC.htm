🔴 Tecnologías de streaming de baja latencia

  1. RTSP / RTP (UDP)

    • Es lo que usan cámaras IP.

    • Muy baja latencia (sub-100 ms).

    • Yolo puede leer directamente un rtsp://... con OpenCV (cv2.VideoCapture).

    • Ideal si desde el móvil puedes emitir RTSP (hay apps en Android/iOS que lo permiten, como Larix Broadcaster).

  2. WebRTC

    • Latencia ultrabaja (10–50 ms).

    • Está pensado justo para transmisión bidireccional en tiempo real.

    • El problema: integrar WebRTC en Python es más complicado (necesitas librerías como aiortc, y aún así no va tan fino como en JS/Go).

    • Si quieres algo universal (abrir en navegador, etc.), es la mejor opción.

  3. SRT (Secure Reliable Transport)

    • Alternativa moderna a RTMP.

    • Diseñado para baja latencia sobre redes inestables.

    • Compatible con FFmpeg → fácil de redirigir a Yolo.

    • Latencia ~100 ms o menos.

  4. RTMP clásico

    • Fácil de montar (con Nginx+RTMP).

    • Pero tiene más lag (~1–3 segundos).

    • Probablemente demasiado para tu caso.

🔧 5. index.html (frontend móvil)

Ejemplo mínimo para capturar la cámara y mandarla al servidor:

html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <title>WebRTC Camera</title>
</head>
<body>
  <h1>Streaming desde el móvil</h1>
  <video id="localVideo" autoplay playsinline></video>

  <script>
    async function start() {
      // Capturar cámara
      const stream = await navigator.mediaDevices.getUserMedia({ video: true, audio: false });
      document.getElementById('localVideo').srcObject = stream;

      // Crear PeerConnection
      const pc = new RTCPeerConnection();
      stream.getTracks().forEach(track => pc.addTrack(track, stream));

      // Crear oferta SDP
      const offer = await pc.createOffer();
      await pc.setLocalDescription(offer);

      // Enviar oferta al backend Go
      const resp = await fetch('/offer', {
        method: 'POST',
        body: JSON.stringify(offer),
        headers: { 'Content-Type': 'application/json' }
      });

      const answer = await resp.json();
      await pc.setRemoteDescription(answer);
    }

    start();
  </script>
</body>
</html>
